# Plan 03-01: Python Bridge

## Objective

Implementar el puente Foundation.Process() que ejecuta el CLI Python de forma asíncrona, captura la salida JSON en background sin bloquear la UI, y decodifica el resultado al modelo ExtractionResult.

## Tasks

### Task 1: Crear PythonBridge.swift con Process + readabilityHandler

Archivo ExtractorApp/ExtractorApp/ExtractorApp/Services/PythonBridge.swift.
Implementar type PythonBridge con método run(url:type:selector:timeout:) async throws -> ExtractionResult.
Foundation.Process() con launchPath, arguments, pipes stdout/stderr.
readabilityHandler en background DispatchQueue para captura sin deadlock.
Timeout implementation.

### Task 2: Decodificar JSON a ExtractionResult

JSONDecoder decodifica stdout capturado a ExtractionResult.
ExtractionError enum: pythonNotFound, processTerminated, jsonDecodeFailed, timeout.
Manejo de stderr, incluir en error message.
Soporte campo "title" (Phase 06).

### Task 3: Task.detached para async/await sin bloquear MainActor

run() es async throws.
Ejecución en Task.detached (background).
@MainActor CallSite puede awaitar sin deadlock.
Cancellation support.

### Task 4: Tests + Manual Verification

Test: run() con CLI válido devuelve ExtractionResult.
Test: run() con URL inválida devuelve ExtractionError.processTerminated.
Test: timeout behavior.
Manual: Llamar desde ContentView, sin congelación UI.
