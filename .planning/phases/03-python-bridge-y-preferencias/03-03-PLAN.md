# Plan 03-03: UI Premium + Branding

## Objective

Rediseñar interfaz visual y crear identidad de marca (logo + icon) con excelencia creativa.

## Tasks

### Task 1: Rediseñar ContentView (UI Premium)

ContentView completamente rewritten: moderno, atrayente.
Layout: Hero (logo/title) → Input (URL + options) → Button → Results.
Colores: paleta cohesionada (primario, secundario, neutrales).
Tipografía: SF Pro, Mono para snippets.
Espaciado: 8px grid, generous margins.
Animaciones sutiles: button states, loading, transitions.
Dark mode full support.
Accessible: contrast ≥4.5:1, keyboard nav, VoiceOver.

### Task 2: Diseñar Logo Vectorial (Marca Visual)

Logo representa data extraction/synthesis.
Estilo minimalist vector (2-3 colores, escalable).
Variantes: full (text + mark), icon-only, monochromatic.
Colores coherentes con UI.
Formats: SVG, PNG (100x100, 256x256).
Distintivo, memorable, profesional.

### Task 3: Diseñar App Icon macOS (Excelencia Visual)

Master 1024px PNG, sharp details.
Estilo: macOS modern (rounded corners, subtle shadow).
Coherente con logo (icon-only mark escalado).
Variantes: light, dark, high contrast.
Xcode asset integration (16-512px).
Verificar: dock muestra correctamente en ambos modos.

### Task 4: Human Verification — Visual Coherence

**Type:** human-verify

Lanzar app desde Xcode (Cmd+R).
Verificar: ContentView new design, responsive, dark mode works.
Logo visible en hero.
App icon en Dock correcto, ambos modos.
Colores: logo + UI + icon coherentes.
Responde: "approved" si excelente; describe issues si las hay.
