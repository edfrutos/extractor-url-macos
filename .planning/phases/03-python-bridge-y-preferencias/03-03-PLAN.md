## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-07 | Tampering | AppStorage pythonPath / scriptPath | mitigate | Validación con FileManager.isExecutableFile() y fileExists() antes de ejecutar; path vacío se rechaza explícitamente |
| T-03-08 | Denial of Service | UI freeze en Task.detached | mitigate | Actualizaciones de @Published siempre dentro de await MainActor.run { }; Task.detached garantiza hilo separado del @MainActor |
| T-03-09 | Information Disclosure | Error messages expuestos en UI | accept | Mensajes de error internos son visibles al usuario — herramienta personal, sin datos sensibles en rutas |
| T-03-SC | Tampering | npm/pip/cargo installs | accept | Este plan no instala paquetes — solo crea ficheros Swift |
</threat_model>
