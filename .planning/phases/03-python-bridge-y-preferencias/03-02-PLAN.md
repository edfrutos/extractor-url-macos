# Plan 03-02: SettingsView Reactivo

## Objective

Crear interfaz Preferencias moderna donde configurar Python y script con validación en tiempo real.

## Tasks

### Task 1: Diseñar SettingsView UI (Premium)

SettingsView.swift con Form/Sections.
2 path input fields: "Python Interpreter", "extractor_url.py Script".
File picker buttons (NSOpenPanel).
Validation indicators (✓ valid, ✗ not found, ⚠ not executable).
Help text, dark mode support.

### Task 2: SettingsViewModel + @AppStorage Binding

SettingsViewModel: @Published paths, validation state.
@AppStorage persistencia: pythonInterpreterPath, scriptPath.
Methods: validatePythonPath(), validateScriptPath() (File.exists + executable).
Reactive updates.

### Task 3: NSOpenPanel Integration + Validation Logic

File picker para cada path.
Validation: fileExists, executable permissions (stat).
Visual feedback: checkmark/X/spinner.
Error messages inline.

### Task 4: Preferences Scene + Tests

ExtractorApp.swift: .preferenceGroup() wiring.
SettingsViewModelTests: path validation, storage.
Manual: set paths, restart, verify persistence.
