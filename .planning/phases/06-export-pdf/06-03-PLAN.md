## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Contenido del webview to PDF Data | El HTML renderizado (de origen remoto) se serializa a PDF |
| PDF Data to disco | Escritura del PDF en una ruta elegida por el usuario |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-06 | Tampering | escritura del PDF a disco | mitigate | `data.write(to:options:.atomic)` (write + rename) evita corrupcion parcial |
| T-06-07 | Information Disclosure | ruta de guardado del PDF | accept | NSSavePanel delega permisos de filesystem al OS; el usuario elige la ruta explicitamente |
| T-06-08 | Denial of Service | createPDF sobre contenido grande | accept | Contenido ya renderizado; sub-segundo; sin entrada externa nueva durante la captura |
| T-06-09 | Tampering | filename derivado de title remoto | mitigate | suggestedFilename (Plan 02) sanea el title antes de usarlo como nombre |
| T-06-SC | Tampering | instalaciones npm/pip/cargo | accept | No se instalan paquetes nuevos (RESEARCH.md Package Legitimacy Audit no aplica) |
</threat_model>
